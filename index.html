RDFa Inspect

This bookmarklet provides a simple way to inspect RDFa embedded in a web page.

Note: for now, if you are attempting to inspect RDFa on a page loaded over HTTPS, by default your browser will block this bookmarklet because it is loading unencrypted content. You will have to use the browser override to allow the bookmarklet to proceed.

Inspect RDFa {% if site.env != "PROD" %}({{ site.env }}){% endif %}

To install the bookmarklet, drag the link to your bookmarks toolbar or right-click and add to your favorites.

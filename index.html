RDFa Utilities

These bookmarklets provide simple ways to inspect RDFa embedded in a web page.

Inspect RDFa {% if site.env != "PROD" %}({{ site.env }}){% endif %} - view a summary of all RDFa on a web page

Highlight RDFa {% if site.env != "PROD" %}({{ site.env }}){% endif %} - highlight sections of a webpage with RDFa and provide tooltip-style information

To install the bookmarklet, drag the link to your bookmarks toolbar or right-click and add to your favorites.
